Tecnologías para crear páginas web

HTML 5

HTML es el lenguaje de creación para las páginas web. Ideado en los inicios de la World Wide Web, se trata del lenguaje de referencia.

Hoy en día se considera a HTMl 5 el conjunto completo de tecnologías relacionados con la creación de páginas web (incluyendo a CSS3, JavaScript y y todos los componentes modernos de creación como WebGL, Web Sockets,...)

Futuro de HTML 5

El futuro está asegurado ya que HTMl 5 no parece tener rival (como en su día ocurrió con XHTML), por lo que tenemos HTML 5 para rato

CSS 3

Las hojas de estilo en casacada o, simplemente, CSS, son uno de los elementos fundamentales en la creación de aplicaciones web actuales. Se trata de un lenguaje en cargado de dar formato a los documentos HTML

Desde su aparición en 1996 ha sufrido múltiples mejoras. Actualmente con CSS 3 el lenguaje sigue creaciendo permitiendo innovadoras habilidades como transiciones, animaciones o maquetaciones avanzadas

CSS y XML

Con XML CSS tiene también un buen comportamiento. Permtie dar formato con muchas posibilidades. No obstante, XSL suele ser el lenguaje habitual para esta tarea ya que permite convertir el documento XML en su equivalente HTML o incluso PDF.

JavaScript

Ideado en 1995 por la compañía Netscape, JavaScript permitió dotar a las páginas web de un verdadero lenguaje que les dotará de mayores prestaciones.

Su crecimiento ha sido voraz, de modo que hacia el año 2001 se hablaba de DHTML hablando de las capacidades dinámicas que JavaScript y CSS daban a las páginas web

AJAX

Microsoft fue la creadora de sta tecnología que permitía que la propia página lanzará peticiones a los servidores minimizando la carga de contenidos en la página.

Hoy en día la inmensa mayoría de páginas utiliza esta técnica intensivamente

jQuery

Se trata de una plantilla que facilita enormemente el trabajo con JavaScript, es el framework más utilizado con diferencia y ha permitido facilitar la compatibilidad y nuevos dinamismos en las páginas.
